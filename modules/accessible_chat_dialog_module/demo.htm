Accessible Chat Dialog Module Demo

The Accessible Chat Dialog Module is a Facebook-style UI component that renders an automatically accessible chat dialog for screen reader and keyboard only users.
Module files:

  1. setup.js : The setup script that configures live chat functionality.

For specific coding guidance, view the AccDC Technical Style Guide

Accessible Chat Dialog Demo

Instructions:

  • Type your name and activate the Start Chat button to open the chat dialog,
    then type into the Message field and press Enter to submit a chat message.
  • New messages will automatically be announced to screen reader users when they arrive.
  • Screen reader users can press Ctrl+` to hear the last 5 messages announced when the Message text field has focus.
    Change Hotkey
  • Keyboard only users can press Shift+Tab from the message input field to set focus on the scrollable message window, which can be controled using the arrow keys, PageUp/Down, and Home/End keys.
